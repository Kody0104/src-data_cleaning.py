# ism2411-data-cleaning-copilot

Mini data-cleaning project for ISM 2411. This repository contains a small Python script that cleans a messy sales dataset and saves a cleaned version for analysis. It is designed to be simple, readable, and something I would be comfortable showing on a resume or to a recruiter.
